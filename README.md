# Feria Digital

Este proyecto es una página web para una feria digital, que incluye un formulario de contacto y elementos interactivos.

## Archivos
- `index.html`: Página principal
- `estilo.css`: Estilos CSS
- `funcion.js`: Funcionalidades JavaScript
- `imagen/`: Carpeta con imágenes

## Instrucciones
1. Abre `index.html` en un navegador.
2. Completa el formulario y envíalo.

## Preguntas y Respuestas

### ¿Qué errores encontró al probar el formulario?
Al probar el formulario, encontré errores como campos vacíos no validados, falta de mensajes de error claros y posibles problemas de envío si no hay JavaScript habilitado.

### ¿Cómo los solucionó?
Agregué validación en JavaScript para verificar campos obligatorios, mostré mensajes de error dinámicos y aseguré que el formulario funcione con y sin JavaScript usando atributos HTML5 como `required`.

### ¿Por qué es importante la estructura semántica?
La estructura semántica mejora la accesibilidad para lectores de pantalla, facilita el SEO y hace el código más legible y mantenible.

### ¿Cómo afecta la jerarquía a la accesibilidad?
Una jerarquía clara (usando encabezados h1-h6) permite a los usuarios con discapacidades navegar mejor el contenido, ya que las herramientas de accesibilidad usan estos niveles para crear un mapa del sitio.

### ¿Qué mejoraría si este proyecto creciera a nivel municipal?
Para escalar a nivel municipal, se podría integrar una base de datos para almacenar datos de usuarios, agregar autenticación, implementar un sistema de gestión de contenido (CMS), optimizar para móviles y agregar análisis de uso.
